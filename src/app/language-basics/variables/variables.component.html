LET and VAR

The main difference between var and let varables declaration is a scoping behaviour.

When variable is declared using let, it uses what is called lexical-scoping or block-scoping. Unlike variables declared with var whose scopes leak out of their containing block, block-scoped variables are not visible outside of their neares containing block.
